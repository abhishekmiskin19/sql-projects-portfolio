# 🎵 Spotify Playlist Analysis (SQL Project)

## 📌 Project Overview
This project analyzes Spotify playlist data using SQL to uncover insights about user behavior, track popularity, and artist performance.

## 🛠️ Tools Used
- MySQL (DBeaver)
- SQL

## 📂 Database Schema
The project includes the following tables:
- Users
- Playlists
- Tracks
- PlaylistTracks

## 🔍 Key Analysis Performed
- User registration trends
- Playlist analysis (track count, duration)
- Track popularity across playlists
- Artist performance insights
- Advanced ranking using SQL window functions

## 📊 Key SQL Concepts Used
- JOIN (INNER JOIN)
- GROUP BY
- Aggregate Functions (COUNT, AVG, SUM)
- Subqueries
- Window Functions (RANK)

## 📁 Project Structure
- dataset → contains schema & data
- queries → contains SQL queries (basic to advanced)

## 🚀 Key Insights
- Identified top playlists with highest track count
- Found most frequently appearing tracks
- Analyzed user engagement based on playlists
- Ranked tracks and playlists using advanced SQL
